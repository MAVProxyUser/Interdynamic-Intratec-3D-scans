# Folder structure
Each folder contains a 3D Scan (.mtl, .obj, and .png). The obj files are zipped to save space. Additionally a STEP file conversion for each obj mesh scan is included. The "TEMPLATE" file is for taking dimensional accuracy measurements vs the real tubes.

# Creality CR-Scan Otter 3D Scanner

All scans were taken with the Creality [Otter](https://creality3dofficial.com/products/creality-cr-scan-otter-3d-scanner)
